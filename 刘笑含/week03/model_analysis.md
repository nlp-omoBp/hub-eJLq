# 模型分析

## 1. BERT

**优点：**

- 能够很好的理解文字意思，并能参考上下文
- 对没有见过的词语，也能根据词向量的相似来进行猜测

**缺点：**

- 使用资源多
- 处理速度比较慢

---

## 2. Prompt

**优点：**

- 不需要过多的训练模型，只需要针对特定的情况对模型进行微调
- 已经有很多的训练数据，所以样本过少的情况下，也能进行很好的预测
- 能处理比较复杂的情况和分类

**缺点：**

- 需要很多处理资源
- 费用较高
- 结果不稳定

---

## 3. Regex

**优点：**

- 处理速度比较快
- 结果确定
- 不需要训练数据

**缺点：**

- 不灵活
- 需要经常维护

---

## 4. TF-IDF

**优点：**

- 处理速度比较快
- 结果确定

**缺点：**

- 对相似词处理不足，依赖训练集
- 如果训练集很小，则预测不是很准确
